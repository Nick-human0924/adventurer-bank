## 🗄️ 数据库结构更新

### 修改 rules 表（添加category字段）

```sql
-- 添加分类字段
ALTER TABLE rules ADD COLUMN IF NOT EXISTS category TEXT DEFAULT 'other';

-- 更新现有规则分类
UPDATE rules SET category = '运动健康' WHERE name LIKE '%运动%' OR name LIKE '%锻炼%';
UPDATE rules SET category = '学习成长' WHERE name LIKE '%作业%' OR name LIKE '%阅读%' OR name LIKE '%学习%';
UPDATE rules SET category = '生活自理' WHERE name LIKE '%整理%' OR name LIKE '%家务%' OR name LIKE '%起床%';
UPDATE rules SET category = '品德社交' WHERE name LIKE '%帮助%' OR name LIKE '%分享%';
```
