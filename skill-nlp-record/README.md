# 行为银行自然语言录入 Skill

自然语言录入儿童行为到行为银行系统。

## 使用方式

直接发送自然语言描述：
- "安哲今天按时起床了"
- "Andrew完成了ABC reading"
- "申安哲自己读了书"

## 支持识别

**孩子昵称：**
- 安哲 / 申安哲 / Andrew / 安德鲁

**行为关键词：**
- ABC reading / 英语 → 读英语
- 读书 → 自己读书
- 收拾 → 主动收拾
- 家务 → 帮助做家务
- 围棋 → 练习围棋
- 钢琴 → 钢琴练习
- 数学 → 数学练习
- 早起 → 早起套餐
- 夸奖 → 得到夸奖
- 运动类：足球、羽毛球、游泳、跳绳、跑步

## 配置

无需额外配置，直接可用。
